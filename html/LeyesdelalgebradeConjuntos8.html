Teoria de Conjuntos

Tema de la clase Leyes del algebra de conjuntos. Problemas que se resuelven con las propiedades y operaciones de conjuntos. .

Objetivo de la clase Conocer y aplicar las leyes del algebra de conjuntos para demostrar la igualdad de los mismos y resolver ejercicios de razonamiento de operación con conjuntos.

Leyes del Algerba de conjuntos

  1. Asociatividad:

    (A∪B)∪C=A∪(B∪C)(Α∪Β)∪C = Α∪(Β∪C)(A∪B)∪C=A∪(B∪C)

    (A∩B)∩C=A∩(B∩C)(A∩Β)∩C = A∩(Β∩C)(A∩B)∩C=A∩(B∩C)

  2. Conmutatividad:

    A∪B=B∪AΑ∪Β = Β∪ΑA∪B=B∪A

    A∩B=B∩AA∩B = B∩AA∩B=B∩A

  3. Distributividad:

    A∪(B∩C)=(A∪B)∩(A∪C)A∪(Β∩C) = (Α∪Β)∩(Α∪C)A∪(B∩C)=(A∪B)∩(A∪C)

    A∩(B∪C)=(A∩B)∪(A∩C)A∩(Β∪C) = (Α∩Β)∪(Α∩C)A∩(B∪C)=(A∩B)∪(A∩C)

  4. Absorción:

    A∪(A∩B)=AA∪(Α∩Β) = ΑA∪(A∩B)=A

    A∩(A∪B)=AA∩(Α∪Β) = AA∩(A∪B)=A

  5. Idempotencia:

    A∪A=AA∪Α = ΑA∪A=A

    B∩B=BB∩Β = ΒB∩B=B

  6. Identidad:

    A∪∅=AΑ ∪ ∅ = ΑA∪∅=A

    A∩U=AΑ ∩ U = AA∩U=A

    A∪U=UA∪U = UA∪U=U

    A∩∅=∅A∩∅ = ∅A∩∅=∅

  7. Complemento:

    A∪Ac=UA∪Α^c = UA∪Ac=U

    A∩Ac=∅A∩Α^c = ∅A∩Ac=∅

    (Ac)c=A(A^c)^c = A(Ac)c=A

    Uc=∅U^c= ∅Uc=∅

    ∅c=U∅^c = U∅c=U

  8. Ley de Morgan:

    (A∪B)c=Ac∩Bc(A∪B)^c = A^c ∩ Β^c(A∪B)c=Ac∩Bc

    (A∩B)c=Ac∪Bc(A∩Β)^c = A^c ∪ Β^c(A∩B)c=Ac∪Bc

    A–B=A∩BcA – B = A ∩ Β^cA–B=A∩Bc

Ejemplos

Demuestre que:

  1. (A∪A)∩(A∪Bc)=A(A ∪ A) ∩ (A ∪ B^c) = A(A∪A)∩(A∪Bc)=A

    Aplicaremos

    (A∪A)∩(A∪Bc)=A(A ∪ A) ∩ (A ∪ B^c) = A(A∪A)∩(A∪Bc)=A Ley de Idempotencia

    A∩(A∪Bc)=AA ∩ (A ∪ B^c) = AA∩(A∪Bc)=A Ley de Absorcicion

    A=AA = AA=A LQQD

  2. (A−B)∩B=∅(Α - B) ∩ Β = ∅(A−B)∩B=∅

    Aplicaremos

    (A−B)∩B=∅(Α - B) ∩ Β = ∅(A−B)∩B=∅ Ley de Morgan

    (A∩Bc)∩B=∅(A ∩ B^c) ∩ B = ∅(A∩Bc)∩B=∅ Ley Asociativa

    A∩(Bc∩B)=∅A∩(B^c ∩ B) = ∅A∩(Bc∩B)=∅ Ley Complemento

    A∩∅=∅Α ∩ ∅ = ∅A∩∅=∅ Ley Identidad

    ∅=∅∅ = ∅∅=∅ LQQD

  3. (B∩C)∪A=(B∪A)∩(C∪A)(B ∩ C) ∪ A = (B ∪ A) ∩ (C ∪ A)(B∩C)∪A=(B∪A)∩(C∪A)

    Aplicaremos

    (B∩C)∪A=(B∪A)∩(C∪A)(B ∩ C) ∪ A = (B ∪ A) ∩ (C ∪ A)(B∩C)∪A=(B∪A)∩(C∪A) Ley Conmutativa

    A∪(B∩C)=(B∪A)∩(C∪A)A ∪ (B ∩ C) = (B ∪ A) ∩ (C ∪ A)A∪(B∩C)=(B∪A)∩(C∪A) Ley Distributiva

    (A∪B)∩(A∪C)=(B∪A)∩(C∪A)( A ∪ B) ∩ (A ∪ C) = (B ∪ A) ∩ (C ∪ A)(A∪B)∩(A∪C)=(B∪A)∩(C∪A) Ley Conmutativa

    (B∪A)∩(C∪A)=(B∪A)∩(C∪A)(B∪A)∩(C∪A) = (B∪A)∩(C∪A)(B∪A)∩(C∪A)=(B∪A)∩(C∪A) LQQD

  4. (A–B)∩(A−C)=A–(B∪C)(A – B) ∩ (Α - C) = A – (B ∪ C)(A–B)∩(A−C)=A–(B∪C)

    Aplicaremos

    (A–B)∩(A−C)=A–(B∪C)(A – B) ∩ (Α - C) = A – (B ∪ C)(A–B)∩(A−C)=A–(B∪C) Ley de Morgan

    (A∩Bc)∩(A∩Cc)=A–(B∪C)(A ∩ Β^c) ∩ (Α ∩ C^c) = A – (B ∪ C)(A∩Bc)∩(A∩Cc)=A–(B∪C) Ley Asociativa

    A∩(Bc∩A)∩Cc=A–(B∪C)A ∩ (B^c ∩ Α ) ∩ C^c = A – (B ∪ C)A∩(Bc∩A)∩Cc=A–(B∪C) Ley Asociativa

    (A∩A)∩(Bc∩Cc)=A–(B∪C)(A ∩ Α) ∩(Β^c ∩ C^c) = A – (B ∪ C)(A∩A)∩(Bc∩Cc)=A–(B∪C) Ley de Idempotencia y Ley de Morgan

    A∩(B∪C)c=A–(B∪C)A ∩ (Β ∪ C)^c = A – (B ∪ C)A∩(B∪C)c=A–(B∪C) Ley de Morgan

    A–(B∪C)=A–(B∪C)A – (B ∪ C) = A – (B ∪ C)A–(B∪C)=A–(B∪C) LQQD

  5. (C∩Ac)∪(B∩Bc)=[C−(A∪B)]∪[(B∩C)−(A∩B∩C)](C ∩ A^c) ∪ (B ∩ B^c) = [C- (A ∪ B)] ∪ [(B ∩ C)- ( A ∩ B ∩ C)](C∩Ac)∪(B∩Bc)=[C−(A∪B)]∪[(B∩C)−(A∩B∩C)]

    Aplicaremos

    (C∩Ac)∪(B∩Bc)=[C−(A∪B)]∪[(B∩C)−(A∩B∩C)](C ∩ A^c) ∪ (B ∩ B^c) = [C- (A ∪ B)] ∪ [(B ∩ C)- ( A ∩ B ∩ C)](C∩Ac)∪(B∩Bc)=[C−(A∪B)]∪[(B∩C)−(A∩B∩C)] Ley Complemento

    (C∩Ac)∪(∅)=[C−(A∪B)]∪[(B∩C)−(A∩(B∩C))](C ∩ A^c) ∪ (∅) = [C- (A ∪ B)] ∪ [(B ∩ C)- ( A ∩ (B ∩ C))](C∩Ac)∪(∅)=[C−(A∪B)]∪[(B∩C)−(A∩(B∩C))] Ley Identidad, de Morgan y Asociativa

    (C∩Ac)=[C∩(A∪B)c]∪[(B∩C)∩(A∩(B∩C))c](C ∩ A^c) = [C ∩ (A ∪ B)^c] ∪ [(B ∩ C) ∩ ( A ∩ (B ∩ C))^c](C∩Ac)=[C∩(A∪B)c]∪[(B∩C)∩(A∩(B∩C))c] Ley de Morgan

    (C−A)=[C∩(Ac∩Bc)]∪[(B∩C)∩(Ac∪(B∩C)c)](C - A) = [C ∩ (A^c ∩ B^c)] ∪ [(B ∩ C) ∩ ( A^c ∪ (B ∩ C)^c)](C−A)=[C∩(Ac∩Bc)]∪[(B∩C)∩(Ac∪(B∩C)c)] Ley Asociativa y Distributiva

    (C−A)=[(C∩Ac)∩Bc)]∪[((B∩C)∩Ac)∪((B∩C)∩(B∩C)c)](C - A) = [(C ∩ A^c) ∩ B^c)] ∪ [((B ∩ C) ∩ A^c ) ∪ ((B ∩ C) ∩ (B ∩ C)^c)](C−A)=[(C∩Ac)∩Bc)]∪[((B∩C)∩Ac)∪((B∩C)∩(B∩C)c)] Ley Complemento

    (C−A)=[(C∩Ac)∩Bc)]∪[((B∩C)∩Ac)∪(∅)](C - A) = [(C ∩ A^c) ∩ B^c)] ∪ [((B ∩ C) ∩ A^c ) ∪ (∅)](C−A)=[(C∩Ac)∩Bc)]∪[((B∩C)∩Ac)∪(∅)] Ley Identidad

    (C−A)=[(C∩Ac)∩Bc)]∪[((B∩C)∩Ac)](C - A) = [(C ∩ A^c) ∩ B^c)] ∪ [((B ∩ C) ∩ A^c )](C−A)=[(C∩Ac)∩Bc)]∪[((B∩C)∩Ac)] Ley Asociativa

    (C−A)=[(C∩Ac)∩Bc)]∪[B∩(C∩Ac)](C - A) = [(C ∩ A^c) ∩ B^c)] ∪ [B ∩ (C ∩ A^c )](C−A)=[(C∩Ac)∩Bc)]∪[B∩(C∩Ac)] Ley Distributiva

    (C−A)=(C∩Ac)∩(B∪Bc)](C - A) = (C ∩ A^c) ∩ (B ∪ B^c )](C−A)=(C∩Ac)∩(B∪Bc)] Ley Complemento

    (C−A)=(C∩Ac)∩(U)](C - A) = (C ∩ A^c) ∩ (U)](C−A)=(C∩Ac)∩(U)] Ley Identidad

    (C−A)=(C∩Ac)(C - A) = (C ∩ A^c)(C−A)=(C∩Ac) Ley de Morgan

    (C−A)=(C−A)(C - A) = (C - A)(C−A)=(C−A) LQQD.
